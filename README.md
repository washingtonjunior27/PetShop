# 🐾 Sistema de Petshop (Em desenvolvimento)

Projeto em desenvolvimento com foco em gerenciamento de petshop.

## 🚀 Tecnologias
- PHP (MVC)
- MySQL
- Bootstrap

## 📌 Status
🔧 Em desenvolvimento

## 💡 Objetivo
Criar um sistema completo para uso interno com:
- Tela de Login
- Alterar senha após primeiro login
- Dashboard
- Cadastro de usuarios(clientes, esteticistas, atendentes e veterinarios)
- Cadastro de pets
- Controle de serviços
- Controle de vacinas
- Agendamentos (Atendente realiza agendamento de clientes)
- Consultas (Atendente realiza confirmação de agendamentos proximos)
- Meus Serviços (Esteticista realiza confirmação de atendimentos)
- Atendimentos (Veterinario realiza confirmação de agendamentos para o mesmo)
- Lembrete (Retornos de vacinas para segunda dose)
- Histórico Medico (Veterinario)
- Historico de Serviço (Esteticista)
